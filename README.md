# vue-client-qna
Q&A Service web 화면입니다.

### 고객문의 접수 & 답변 개발 화면 입니다.

### 1. 프로젝트 구성(Client)

* UI Framework : Vue.js(ver.2)
* IDE Tool : Visual Studio Code
---
### 2. 실행 방법

1. Source Code Download
```
git clone 
```
2. node module 설치
```
cd vue-client-cs
npm install
```
3. 실행
```
npm run serve
```
4. 접속
```
chrome 에서 http://localhost:8080 으로 접속
```
5. 단위테스트
```
npm run test:unit
```
